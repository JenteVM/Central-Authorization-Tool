# Security Policy

This is an hobby project of mine. There may be big security flaws because of that. Any present flaws that have been reported will be fixed but support may not be the fastest. I made this project as a way to have simple security, anything but low security applications should never use this software.

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.3.x   | 🟩 |
| < 0.3.0   | 🟥 |

## Reporting a Vulnerability

For reporting on any Vulnerability, please use the security flaws scheme. I will try and fix the problem for the next patch, but it may take a while.
